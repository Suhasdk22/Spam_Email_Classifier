# Spam_Email_Classifier
An NLP-based spam classifier that detects spam messages using text preprocessing, TF-IDF vectorization, and Naive Bayes / Logistic Regression models. Trained on the SMS Spam Collection dataset (5,572 messages) in Google Colab, achieving 97% accuracy with full evaluation and live message testing.

Spam Email Classifier (NLP)

Python, Scikit-learn, NLP (TF-IDF, text preprocessing), Naive Bayes, Logistic Regression

Built a spam detection system that classifies messages as spam or not spam, achieving 97% accuracy on a 5,572-message labeled dataset.
